记事本太轻 WPS太重

发表于 2019-06-23 | 分类于 Basic

2346个词，预计阅读5分钟。

最不完整的 Markdown 介绍，顺便推荐几款好用的 Markdown 编辑器。

Markdown 是什么？

首先，你看到的这篇文章是用电脑写的。

其次，你看到的这篇是先用文本编辑器写好，然后粘贴到微信公众号的。

最后，我使用的是 Markdown 编辑器。

由此可得，Markdown 编辑器是一种帮助编辑文字的东东。

Markdown is a text-to-HTML conversion tool for web writers. Markdown allows you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid XHTML (or HTML).

”Markdown是一种轻量级标记语言，创始人是约翰·格鲁伯（John Gruber）。它允许人们 “使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。”

“易读易写”，文思喷涌的你只需要花几分钟入门就可以使用 Markdown 来写文章了。

为什么要用 Markdown？

你看这篇文章，不仅有充实的内容，而且还有华丽的排版。充实的内容使你读完颇有收获，华丽的排版让你的阅读轻松愉快。

如果你使用记事本来写这篇文章，（记事本是最简单的文本编辑器，windows自带），你只能拥有充实的内容。而如果你使用WPS，（WPS文字，一种办公用的文字处理软件，功能强大），你既可以拥有充实的内容，也还能拥有华丽的排版，但是你不得不在键盘和鼠标之间来回切换，以便输入文字并调整格式，甚至有时候为了一个小小的缩进大费周折。这对于文思喷涌的你来说这简直是一个灾难，犹如云水边静沐暖阳的你，身披战甲。

而有了 Markdown，你就可以换上人字拖大裤衩 polo 衫了！

Markdown 怎么用 之编辑器

显然，我这里指的是 Markdown 编辑器。

在我安利之前，你可以不带任何主观色彩地看下这篇文章：12 款 Markdown 写作工具推荐。

好了，身为资深 Markdown 使用者的我，现在隆重地给你推荐几款工具，亲测好用：

  1. Typora

    Typora 是我近期刚开始用的 Markdown 编辑器，六个字，真的非常好用；五个字，所见即所得；四个字，相见恨晚；三个字，有格调；两个字，舒服；一个字，爽；零个字。

    Typora will give you a seamless experience as

    Typora will give you a seamless experience as both a reader and a writer. It removes the preview window, mode switcher, syntax symbols of markdown source code, and all other unnecessary distractions. Replace them with a real live preview feature to help you concentrate on the content itself.

  2. Visual Studio Code

    Visual Studio Code 微软背书、插件丰富、界面优美。VS Code 原生支持 Markdown 语法，下载之后立刻就可以使用。

    Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux.

  3. Cmd Markdown

    Cmd Markdown 我主要使用在线版本，除此之外，它还有全平台的客户端。

    我们理解您需要更便捷更高效的工具记录思想，整理笔记、知识，并将其中承载的价值传播给他人，Cmd Markdown是我们给出的答案 —— 我们为记录思想和分享知识提供更专业的工具。 您可以使用 Cmd Markdown

写在最后

  • Markdown 把文章的样式字符化，相当于把文章样式的编辑和显示进行了分离，让我们更加专注于文字，使得写作更加流畅。

  • Markdown 文章几乎可以在任何一台个人电脑上完美展示，你不需要担心对方无法浏览，因为只需要一个浏览器就可阅读。如果你想更加方便与他人分享自己的文章，Markdown 就是你想要的。

  • 如果你有一些前端编程基础，Markdown 基于 HTML 的特性会让你有更多手段进行文章的展示。

  • 使用 Markdown 使得文章的版本控制)更加方便，如果你对此感兴趣，你需要了解一下git。
